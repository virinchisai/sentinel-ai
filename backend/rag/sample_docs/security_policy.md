# Security Policy

## Password Requirements
All employees must use passwords of at least 12 characters, including uppercase,
lowercase, numbers, and at least one special character. Passwords must be rotated
every 90 days via the SSO portal.

## Multi-Factor Authentication
MFA is required for all internal systems. Approved methods include hardware security
keys (YubiKey), authenticator apps (Google Authenticator, Authy), and SMS as a
last-resort fallback. Hardware keys are provided upon request from IT.

## Data Classification
- **Confidential**: Customer PII, financial records, API keys, credentials
- **Internal**: Architecture docs, roadmaps, internal tools documentation
- **Public**: Marketing materials, open-source project docs, blog posts

Confidential data must never be stored in unencrypted form or transmitted over
unencrypted channels. Violations are reported to the CISO within 24 hours.
